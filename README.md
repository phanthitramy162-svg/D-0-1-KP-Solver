# D{0-1}KP 智能求解系统 
## 项目简介 
- 可正确读入实验数据文件的有效D{0-1}KP数据
- 能够绘制任意一组D{0-1}KP数据以重量为横轴、价值为纵轴的数据散点图
- 能够对一组D{0-1}KP数据按项集第三项的价值:重量比进行非递增排序
- 用户能够自主选择指定D{0-1} KP数据的获得最优求解结果
- 任意一组D{0-1} KP数据的最优解、求解时间和解结果可保存为txt文件或导出EXCEL文件
- 支持动态规划与贪心算法性能对比，并绘制相关柱状图
- 一键导出带有排序结果的数据报表
- 提供一个友好的用户界面
- 用户可以自定义创建数据集，并进行求解
- 搭载防无响应多线程引擎
